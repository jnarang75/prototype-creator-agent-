‚ùì Frequently Asked Questions
‚ñº
1. How do I install Claude CLI?
Visit the Claude Code installation guide on our SharePoint site for step-by-step instructions.
2. Why do I need to copy prompt to Claude CLI?
Claude CLI runs locally on your machine and has access to your file system to create prototypes, read mockup files, and open results in your browser. The web version cannot perform these local file operations.
3. How can I iterate over design?
Simply tell Claude what changes you want! For example: "Make the QR code smaller", "Change the background color to match our brand", or "Add another scenario". Claude will update the prototype based on your feedback.
4. How can I host my prototype so everyone can access?
Upload the HTML file to OneDrive, SharePoint, or GitHub Pages. Share the link with your team. Since it's a single HTML file, anyone can download and open it directly in their browser.
5. Who should I contact for feedback on this agent?
Email your feedback, suggestions, or questions to rapatha@microsoft.com. We'd love to hear how this tool is working for you!
